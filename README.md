# prenotazioni-uffici-
Sistema web per la gestione e prenotazione di risorse condivise (sale riunioni, postazioni e attrezzature), sviluppato con PHP, MySQL e JavaScript. L’applicazione consente agli utenti di verificare la disponibilità in tempo reale, evitare conflitti tramite controlli automatici e lock temporanei, gestire le proprie prenotazioni e accedere a report
